# Wedding Comomeer

Website for Dylan and Susan's wedding at Lake Como, Italy.

## Sections

- Agenda: Schedule for the weekend
- Location: Information about Agriturismo Castello di Vezio with images
- Preparations: What to bring and prepare
- FAQ: Frequently asked questions
- Menu: Food menu for the events
- Drinks: Drink menu

## Technologies

- Vite
- Vanilla JavaScript
- CSS

## How to run locally

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Open http://localhost:5173 in your browser

## Build for production

`npm run build`

The built files will be in the `dist` folder, ready to deploy to GitHub Pages or any static host.

## Deployment

Push to GitHub and enable GitHub Pages from the `dist` folder.
